## Screenshot Guidelines

- Use high resolution (1920x1080 or higher)
- Ensure good lighting and contrast
- Include the full page in the screenshot
- Make sure all text is readable
- Use PNG format for best quality
- Name files exactly as specified above

## Tools for Screenshots

- **Desktop**: Use browser dev tools (F12) or screenshot tools
- **Mobile**: Use browser responsive design mode
- **Full Page**: Use browser extensions like "Full Page Screen Capture"
